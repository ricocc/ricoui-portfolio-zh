# 零基础新手网站开发入门教程

## 概述

本教程面向完全没有编程基础的初学者，将手把手教你如何使用 Claude Code + Kimi 创建一个现代化的网站，并部署到互联网上。

## 教程目录

1. [准备工作](./01-准备工作.md) - 账号注册和必备工具
2. [环境安装](./02-环境安装.md) - Node.js、npm 等开发环境配置
3. [IDE 使用指南](./03-IDE使用指南.md) - 使用 Trae AI 辅助开发
4. [项目启动与运行](./04-项目启动与运行.md) - 本地开发和测试
5. [网站内容修改](./05-网站内容修改.md) - 修改文字、图片等基础内容
6. [主题定制](./06-主题定制.md) - 使用 Figma 设计系统自定义颜色和样式
7. [部署上线](./07-部署上线.md) - 发布到 Netlify、Cloudflare、Vercel

## 你将学会

- ✅ 注册必要的开发账号（GitHub、Vercel 等）
- ✅ 安装和配置开发环境
- ✅ 使用 AI 辅助编程工具 Trae
- ✅ 修改网站内容和样式
- ✅ 使用 Figma 设计系统管理颜色和组件
- ✅ 将网站部署到互联网上

## 学习目标

完成本教程后，你将能够：
1. 独立搭建一个现代化的个人网站
2. 使用 AI 工具辅助开发
3. 自定义网站的外观和内容
4. 将网站发布到互联网供他人访问

## 开始之前

请确保你有：
- 一台可以上网的电脑
- 基本的电脑操作能力
- 学习新事物的热情

让我们开始这个激动人心的旅程吧！

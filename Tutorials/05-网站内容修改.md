# 网站内容修改

## 了解你的网站

### 常见文件结构

```
网站项目/
├── public/                    # 公共资源
│   ├── images/               # 图片文件夹
│   └── favicon.ico          # 网站图标
├── src/
│   ├── components/           # 组件（可复用的部分）
│   │   ├── Header.vue       # 网站头部
│   │   ├── Footer.vue       # 网站底部
│   │   └── Button.vue       # 按钮组件
│   ├── pages/               # 页面
│   │   ├── Home.vue         # 首页
│   │   ├── About.vue        # 关于页面
│   │   └── Contact.vue      # 联系页面
│   ├── styles/              # 样式文件
│   │   ├── main.css         # 主样式
│   │   └── variables.css    # 颜色变量
│   └── assets/              # 项目资源
├── package.json             # 项目配置
└── README.md               # 项目说明
```

## 修改基础内容

### 1. 修改文字内容

**找到对应的文件：**
```html
<!-- 修改前 -->
<h1>欢迎来到我的网站</h1>
<p>这是一个示例网站</p>

<!-- 修改后 -->
<h1>欢迎来到我的个人主页</h1>
<p>我是一名前端开发者，热爱创造美好的用户体验</p>
```

### 2. 修改图片

**AI 助手对话示例：**
```
用户："我想把首页的头像换成我自己的照片，应该怎么做？"

AI 助手：我来帮你替换头像图片。

步骤：
1. 把你的照片放到 public/images/ 文件夹
2. 找到头像的代码位置
3. 修改图片路径

具体代码修改：
```html
<!-- 原来的代码 -->
<img src="/images/default-avatar.jpg" alt="头像" >

<!-- 修改后的代码 -->
<img src="/images/my-photo.jpg" alt="我的头像" >
```

注意：
- 图片文件名用英文
- 图片大小建议不超过 500KB
- 用 .jpg 或 .png 格式
```

### 3. 修改链接

```html
<!-- 修改导航链接 -->
<nav>
  <a href="/">首页</a>
  <a href="/about">关于我</a>
  <a href="/projects">项目作品</a>
  <a href="/contact">联系我</a>
</nav>
```

## 内容管理最佳实践

### 1. 内容组织

```
建议的文件命名：
✅ about-me.html
✅ contact-info.html
✅ project-list.html

❌ about.html
❌ 1.html
❌ test.html
```

### 2. 图片管理

```
图片文件夹结构：
images/
├── avatar/          # 头像图片
├── projects/        # 项目截图
├── backgrounds/     # 背景图片
└── icons/          # 图标
```

### 3. 文本内容规范

```html
<!-- 好的做法：语义化标签 -->
<article>
  <header>
    <h1>文章标题</h1>
    <time>2024-01-01</time>
  </header>
  <section>
    <p>文章内容... </p>
  </section>
</article>

<!-- 不好的做法：全用 div -->
<div class="article">
  <div class="title">文章标题</div>
  <div class="date">2024-01-01</div>
  <div class="content">文章内容... </div>
</div>
```

## 常见错误和解决方案

### 错误 1：图片不显示

**原因：**
- 路径错误
- 文件名错误
- 大小写不匹配

**解决方案：**
```html
<!-- 检查路径是否正确 -->
<img src="/images/my-photo.jpg" alt="我的照片">

<!-- 确认文件存在 -->
<!-- 检查大小写 -->
```

### 错误 2：中文字符乱码

**解决方案：**
```html
<!-- 确保文件编码是 UTF-8 -->
<meta charset="UTF-8">
```

### 错误 3：链接跳转错误

**检查：**
```html
<!-- 正确的相对路径 -->
<a href="/about">关于</a>

<!-- 绝对路径（不推荐） -->
<a href="C:/Users/...">关于</a>  ❌
```

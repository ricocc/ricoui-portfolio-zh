# 环境安装 - 配置开发环境

## 什么是开发环境？

开发环境就像厨师的厨房，需要各种工具和调料。我们的"厨房"需要安装 Node.js、npm 等工具。

## 安装步骤

### 1. 安装 Node.js

**什么是 Node.js？**
Node.js 是运行 JavaScript 代码的环境，就像浏览器一样，但更适合开发。

**检查是否已安装：**
1. 按 `Win + R`（Windows）或打开终端（Mac）
2. 输入 `node --version`
3. 如果显示版本号（如 v18.17.0），说明已安装
4. 如果提示 "找不到命令"，需要安装

**安装步骤：**
1. 打开 [nodejs.org](https://nodejs.org)
2. 下载左边的 LTS 版本（推荐版本）
3. 运行安装程序
4. 一直点击 "下一步"，保持默认设置
5. 安装完成后重启电脑

**验证安装：**
```bash
node --version
npm --version
```

### 2. 安装 Git

**什么是 Git？**
Git 是版本控制工具，可以记录代码的修改历史。

**检查是否已安装：**
```bash
git --version
```

**安装步骤：**
1. 打开 [git-scm.com](https://git-scm.com)
2. 下载适合你系统的版本
3. 安装时保持默认设置
4. 安装完成后重启 Trae

### 3. 安装包管理器（可选但推荐）

#### Windows 用户：安装 Chocolatey
1. 以管理员身份打开 PowerShell
2. 运行：
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```

#### Mac 用户：安装 Homebrew
1. 打开终端
2. 运行：
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 4. 配置开发环境

#### 配置 Git（重要）
打开 Trae 的终端，输入：
```bash
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"
```

#### 配置 npm 镜像（加速下载）
```bash
npm config set registry https://registry.npmmirror.com
```

## 验证安装

在 Trae 的终端中运行以下命令，确保所有工具都安装成功：

```bash
# 检查 Node.js
node --version

# 检查 npm
npm --version

# 检查 Git
git --version

# 检查包管理器（如果安装了）
# Windows:
choco --version
# Mac:
brew --version
```

## 常见问题解决

### 问题 1：npm 安装很慢
**解决方案：**
```bash
npm config set registry https://registry.npmmirror.com
```

### 问题 2：权限错误
**解决方案（Windows）：**
以管理员身份运行 PowerShell

**解决方案（Mac）：**
在命令前加 `sudo`

### 问题 3：找不到命令
**解决方案：**
1. 重启 Trae
2. 检查环境变量
3. 重新安装该工具

## 小贴士

1. **版本问题**：如果安装过程中提示版本冲突，优先选择推荐版本
2. **路径问题**：安装路径不要包含中文和空格
3. **防火墙**：安装时可能需要关闭防火墙
4. **重启**：安装完成后建议重启电脑和 Trae

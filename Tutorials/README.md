# 零基础新手网站开发入门教程

## 📚 教程简介

本教程专为**完全没有编程基础**的新手设计，将手把手教你如何使用 **Claude Code + Kimi** 创建现代化的个人网站，并成功部署到互联网上。

## 🎯 学习目标

完成本教程后，你将能够：
- ✅ 独立搭建现代化的个人网站
- ✅ 使用 AI 工具辅助开发
- ✅ 自定义网站的外观和内容
- ✅ 将网站发布到互联网供他人访问
- ✅ 使用 Figma 设计系统管理颜色和组件

## 🚀 快速开始

如果你是第一次接触网站开发，建议按顺序学习。每节课都包含：
- 📖 **概念解释** - 用简单易懂的方式讲解
- 🛠️ **实操步骤** - 详细的手把手教学
- 🤖 **AI 协作** - 如何使用 Claude Code + Kimi 辅助开发
- 💡 **实用技巧** - 专业开发者的经验分享
- 📝 **课后作业** - 巩固所学知识

## 🎨 特色功能

### AI 辅助开发
- 使用 **Trae** 的智能代码补全
- 利用 **Claude Code** 解决技术问题
- 通过 **Kimi** 优化设计和内容

### Figma 设计集成
- 学习使用 Figma 创建设计系统
- 从设计到代码的完整工作流
- 颜色系统和组件库管理

### 现代化部署
- 一键部署到全球 CDN
- 自动 HTTPS 和性能优化
- 支持自定义域名

## 🛠️ 技术栈

本教程使用现代化的技术栈：
- **前端框架**: React/Vue/Astro（根据项目而定）
- **样式**: CSS3 / SCSS / Tailwind CSS
- **构建工具**: Vite / Webpack
- **部署平台**: Vercel / Netlify / Cloudflare Pages
- **设计工具**: Figma
- **开发环境**: Node.js + Git

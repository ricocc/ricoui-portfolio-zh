# 准备工作 - 注册必要的账号

在开始网站开发之前，我们需要准备一些必要的账号和工具。就像开店需要营业执照一样，做网站也需要一些"证件"。

## 必备账号清单

### 1. GitHub 账号（代码托管）

**什么是 GitHub？**
GitHub 就像一个云端的代码仓库，可以保存你的代码，还能和其他人协作。

**注册步骤：**
1. 打开 [github.com](https://github.com)
2. 点击右上角的 "Sign up"
3. 输入邮箱地址（建议使用常用邮箱）
4. 设置密码（建议使用强密码）
5. 输入用户名（这是你在 GitHub 上的名字，可以修改）
6. 完成邮箱验证

**重要提醒：**
- 记住你的用户名和密码
- 验证邮箱是必须的步骤

### 2. 部署平台账号（任选其一）

#### 选项 A：Vercel（推荐新手）
- 打开 [vercel.com](https://vercel.com)
- 点击 "Sign Up"
- 选择 "Continue with GitHub"（用 GitHub 账号登录）
- 授权 Vercel 访问你的 GitHub

#### 选项 B：Netlify
- 打开 [netlify.com](https://netlify.com)
- 点击 "Sign up"
- 选择 "GitHub" 登录方式
- 授权 Netlify 访问你的 GitHub

#### 选项 C：Cloudflare Pages
- 打开 [cloudflare.com](https://cloudflare.com)
- 点击 "Sign up"
- 填写信息注册
- 需要验证邮箱

**建议：** 先注册 Vercel，因为它最简单，适合新手。

### 3. Figma 账号（设计协作）
**什么是 Figma？**
Figma 是一个在线设计工具，我们用它来管理网站的颜色和组件设计。

**注册步骤：**
1. 打开 [figma.com](https://figma.com)
2. 点击 "Sign up"
3. 可以用 Google 账号或邮箱注册
4. 验证邮箱

## 开发工具准备

### 1. 浏览器（推荐 Chrome）
- 下载 [Google Chrome](https://www.google.com/chrome/)
- Chrome 的开发者工具很好用

### 2. 文本编辑器（Trae - 带 AI 功能）
**什么是 Trae？**
Trae 是一个智能的代码编辑器，内置 AI 助手，可以帮你写代码、改代码、解释代码。

**下载地址：**
- 打开 [trae.ai](https://trae.ai)
- 选择适合你系统的版本下载
- 安装过程很简单，一直点 "下一步" 即可

## 常见问题

**Q: 为什么要注册这么多账号？**
A: 就像开店需要不同的证件一样，每个平台都有它的用途：
- GitHub：保存代码
- Vercel：发布网站
- Figma：设计界面

**Q: 这些账号都免费吗？**
A: 是的，我们使用的都是免费版本，对个人项目完全够用。

**Q: 忘记了密码怎么办？**
A: 每个网站都有 "忘记密码" 功能，通过邮箱可以重置密码。建议用密码管理器保存密码。
